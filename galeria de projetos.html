Projeto Arduino

A Helena com apenas 12 anos desenvolveu um projeto de programação super importante para ajudar as pessoas. O projeto foi feito com placas de arduino, e tem como objetivo lembrar as pessoas de trocar as máscaras. Arduino é um tipo de prototipagem eletrônica. Ou seja, é uma plataforma de desenvolvimento de projetos, tanto de hardware como em software.

Projeto Scratch

A Teresa tem 8 anos e com o estudo da programação ela já cria projetos incríveis e desenvolve jogos completos. O Scratch é uma linguagem de programação que foi desenvolvida na Universidade americana MIT, pelo grupo Lifelong Kindergarten. Essa plataforma é muito utilizada para ensinar tecnologia para crianças e adolescentes, afinal é um tipo de programação “visual” mais simplificada, que permite que os pequenos consigam criar projetos digitais com mais facilidade.

Projeto Kodu

Antonio de 8 anos já é um criador de tecnologia e desenvolveu um projeto de programação. O Kodu Lab Game é um software de criação de jogos desenvolvido pela Microsoft para a plataforma Windows. Ele foi projetado para permitir que crianças e jovens criem seus próprios jogos de forma fácil e intuitiva. Por isso, a plataforma conta com uma interface super visual com ícones para permitir que os usuários criem cenários, personagens, objetos e eventos que acontecem no jogo.
